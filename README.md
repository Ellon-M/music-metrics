# 📖 Music Metrics <a name="about-project"></a>

A web application displaying featured Spotify playlists based on the Spotify algorithm selection and shows details of the most popular tracks in each playlist from the most popular to the least popular by displaying a popularity index value - generated by the number of streams and plays of the track within the recent past.

## 🛠 Built With <a name="built-with"></a>

- React 
- Redux

### Tech Stack <a name="tech-stack"></a>

- HTML

- ReactJS

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.


### Prerequisites

In order to run this project you need:

-NodeJS installed.
-Git installed.
-A web browser of choice.

### Setup

Clone this repository to your desired folder:
 
- Use the `git clone` command with this [link](https://github.com/Ellon-M/music-metrics.git)
- On the terminal, inside the desired folder's directory, run `npm install` to install all the necessary depandancies.
- run `npm start` to execute the code and serve it on the browser.

### Install

- Install this project with:

```
$ npm install
```

### Usage

To run the project, execute the following command:

```
npm run dev
```

### Run tests

To run tests, run the following command:

```
npm run test
```
